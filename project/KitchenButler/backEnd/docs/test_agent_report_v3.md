### 问题 1: 文件未关闭警告

**现象**: 测试执行完毕后出现资源警告

```
ResourceWarning: unclosed file <_io.TextIOWrapper name='data/response.json' mode='w' encoding='utf-8'>
```

**位置**: `app/test/testAgent.py`, line 149

**代码问题**:
```python
json.dump(recipes, open('data/response.json', 'w', encoding="utf-8"), ensure_ascii=False, indent=4)
```

**建议解决方案**:
```python
with open('data/response.json', 'w', encoding="utf-8") as f:
    json.dump(recipes, f, ensure_ascii=False, indent=4)
```
