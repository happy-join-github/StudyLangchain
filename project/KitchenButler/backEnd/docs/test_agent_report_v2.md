## 系统提示词配置

**文件位置**: `app/agent/manage.py`

```python
sys_prompt = """你是一个识别食材的机器人。根据用户上传的图片，识别出图片中的食材。

**重要：请严格按照以下格式输出，不要添加任何其他内容**
**只输出食材名称，每行一个，不要加任何标记符号，不要加英文注释**
**示例输出格式：**
土豆
茄子
胡萝卜
番茄
五花肉
大蒜
月桂叶

请识别图片中的食材："""
```

**提示词设计要点**:
1. 明确角色定位：识别食材的机器人
2. 强调输出格式：每行一个食材名称
3. 禁止添加标记：不使用 `-`、`*` 等列表标记
4. 禁止添加注释：不输出英文名称如 `(Potatoes)`
5. 提供示例输出：让模型更清楚地理解期望格式

## 问题与改进建议

### 问题1：食材识别输出格式不符预期

**现象**: 模型仍输出 Markdown 格式和分类信息，未严格遵守系统提示词。

**可能原因**:
1. 模型对复杂提示词的理解能力有限
2. 示例输出不够充分

**改进建议**:
```python
sys_prompt = """你是一个识别食材的机器人。请看图识别食材。

输出格式要求：
1. 只输出食材名称
2. 每行一个
3. 不要任何标记符号、分类、括号注释
4. 只输出中文食材名

示例：
土豆
茄子
胡萝卜
番茄
五花肉
大蒜
月桂叶"""
```

### 问题2：搜索查询语句过于复杂

**现象**: 搜索查询包含了完整的 Markdown 格式食材列表。

**改进建议**: 在调用搜索前，添加预处理步骤提取纯食材名称：

```python
# 提取食材名称（简单示例）
import re

# 移除 Markdown 格式
ingredients_text = re.sub(r'[*_#\[\]]', '', ingredients_text)
```
